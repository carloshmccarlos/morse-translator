MorseAI — Audio to Morse Code Translator (AI-Powered)

What is MorseAI?

MorseAI is a free online AI-powered Morse code translator that converts spoken words and audio files into Morse code instantly. Upload a WAV, MP3, M4A, or OGG audio file, or record your voice directly in the browser using your microphone. MorseAI uses AI speech recognition to transcribe your audio to text, then automatically converts it to international Morse code (dots and dashes) following the ITU-R M.1677 standard.

Unlike basic text-to-Morse tools, MorseAI supports the full workflow: voice input → AI transcription → Morse output → audio playback → visual lamp display → export. It also supports batch translation of entire files (.txt, .csv, .xlsx) and reverse translation from Morse back to text.

How to Convert Voice to Morse Code — Step by Step

  1. Upload or Record: Click "Select Audio File" to upload a WAV, MP3, M4A, or OGG file (max 3MB), or click "Start Record" to use your microphone (up to 30 seconds).
  2. Submit for AI Transcription: Click "Submit Recording". MorseAI's AI engine (powered by Deepgram) transcribes your spoken words into text.
  3. Translate to Morse: Click "Translate to Morse" to convert the transcribed text into international Morse code.
  4. Play & Export: Click "Play" to hear the Morse tone, adjust WPM speed (5–40), blink it as a visual lamp signal, or download as a .wav audio file. Export as TXT, CSV, or Excel.

Key Features

  • AI Speech-to-Morse: Record your voice or upload an audio file — AI transcribes speech and converts to Morse code automatically.
  • Audio Playback: Hear Morse code as a sine-wave tone at adjustable WPM (words per minute) speed.
  • WAV Audio Export: Download the Morse code signal as a standard WAV audio file.
  • Visual Signal Lamp: Watch Morse code blink as a light signal for visual learning.
  • Batch Translation: Import a .txt, .csv, .xls, or .xlsx file and translate all rows at once. Export results.
  • Reverse Translation: Paste Morse code (dots, dashes, slashes) to decode back to plain text.
  • Learning Mode: Practice decoding random Morse code characters with instant feedback.
  • Full Punctuation: Supports all ITU-R standard punctuation: . , ? ! / ( ) & : ; = + - _ " @ '

Frequently Asked Questions

Is MorseAI free to use?

Yes. MorseAI is completely free with no registration or account required. All features are available at no cost.

What audio formats does MorseAI support?

MorseAI accepts WAV, MP3, M4A, and OGG audio files up to 3MB. You can also record directly with your browser microphone for up to 30 seconds.

Can I decode Morse code back to text?

Yes. Use the Reverse Translation feature: paste Morse code using dots (.), dashes (-), and slash (/) word separators, then click "Reverse Translate Morse" to get the original text.

What Morse code standard does MorseAI use?

MorseAI uses the ITU-R M.1677 International Morse Code standard, covering all 26 letters (A–Z), digits 0–9, and 17 punctuation symbols.

How accurate is the AI speech recognition?

MorseAI uses Deepgram's Nova-2 model, one of the most accurate speech recognition systems available. Accuracy depends on audio quality — clear recordings in quiet environments work best.

What is the WPM setting?

WPM (Words Per Minute) controls Morse tone playback speed. Range: 5 WPM (beginner speed) to 40 WPM (experienced operator speed). The standard learning speed is around 5–13 WPM; amateur radio exams typically use 5–20 WPM.

Can I translate multiple texts at once?

Yes. The Batch Translation Deck feature accepts .txt, .csv, .xls, or .xlsx files. Each paragraph or row is translated to its own Morse entry. Export all results as TXT, CSV, or Excel.
